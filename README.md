# 🌅 星际黎明 (Starlight Dawn)

一款类似EVE Online的AstrBot聊天挂机游戏插件。

> 在浩瀚星海中建立你的帝国，从一名普通飞行员成长为星际传奇。

## 游戏特色

- 🌌 **星系探索**：太阳系8大行星 + 外部非安全区星系
- ⛏️ **挂机挖矿**：安全区7种矿物，非安全区20种矿物
- ⚔️ **战斗刷怪**：1-6级怪物，从游寇到海盗
- 🔧 **舰船制造**：8种舰船，从护卫舰到货舰
- 📚 **技能系统**：17个技能，10级上限，挂机学习
- 👥 **多克隆体**：单玩家可操作多个克隆体组建舰队
- 💰 **贸易系统**：市场交易 + 合同交易

## 安装

1. 将插件文件夹复制到 AstrBot 的插件目录
2. 重启 AstrBot 或重载插件
3. 发送 `/注册` 开始游戏

## 指令列表

### 基础指令
- `/太空游戏` - 显示游戏帮助
- `/注册` - 创建游戏角色
- `/状态` - 查看当前状态
- `/钱包` - 查看资金
- `/资产` - 查看行星资产

### 挖矿系统
- `/挖矿 <行星>` - 开始挖矿（如：/挖矿 地球）
- `/停止挖矿` - 停止挖矿并结算

### 战斗系统
- `/刷怪 <等级>` - 开始刷怪（1-6级）
- `/停止刷怪` - 停止刷怪并结算赏金

### 技能系统
- `/技能` - 查看所有技能
- `/学习 <技能名>` - 学习技能
- `/停止学习` - 停止学习并结算进度

### 制造系统
- `/制造 <舰船> [数量]` - 制造舰船
- `/制造状态` - 查看制造进度

### 舰船系统
- `/舰船` - 查看可用舰船
- `/换船 <舰船名>` - 更换舰船

### 贸易系统
- `/市场` - 查看市场（开发中）
- `/合同` - 创建合同（开发中）

## 游戏机制

### 技能学习
- 每分钟获得20技能点
- 每级所需技能点翻倍（1级=64, 2级=128...）
- 单技能满级约27.3小时
- 全技能满级约19天

### 技能加成公式
```
加成后数值 = 基础值 × (1 + 技能等级 × 10%)
```

### 制造时间
```
制造时间 = 16秒 × (舰船成本 / ¥150万)
```

### 刷怪收益
```
击杀时间 = 怪物血量 ÷ 舰队DPS
总击杀数 = 刷怪时长 ÷ 击杀时间
总赏金 = 总击杀数 × 单只赏金
```
