# 🚀 星际黎明 (Star Dawn)

一款类似 EVE Online 的 AstrBot 太空挂机游戏插件。

> 在浩瀚星海中建立你的帝国，从一名普通指挥官成长为星际传奇。

## 游戏特色

- 🌌 **星系探索**：太阳系8大行星，自由跃迁
- ⛏️ **挂机挖矿**：安全区7种矿物，矿舱容量限制，自动卸货
- 📚 **技能系统**：17个技能，10级上限，挂机学习，进度保留
- 👥 **多克隆体**：单玩家可操作多个克隆体组建舰队
- 🚀 **跃迁系统**：每条舰船独立跃迁速度
- 🚛 **运输系统**：运输船货仓管理，批量装载/卸载
- 💰 **资产系统**：各行星独立存储矿物和舰船

## 安装

1. 将插件文件夹复制到 AstrBot 的插件目录
2. 重启 AstrBot 或重载插件
3. 发送 `/游戏注册` 开始游戏

## 游戏机制

### 初始设置
- 出生点：地球
- 初始舰船：蜜蜂级采矿护卫舰
- 初始资金：¥0
- 初始技能：采矿护卫舰操控理论 Lv.1

### 技能学习
- 学习方式：挂机，每分钟20技能点
- 升级需求：每级翻倍（1级=64点, 2级=128点...）
- 进度保留：切换技能时进度保留，可随时切换
- 单技能满级：约27.3小时
- 全技能满级：约464小时（19天）

### 技能加成公式
```
加成后数值 = 基础值 × (1 + 技能等级 × 10%)
```

### 跃迁机制
```
跃迁时间(分钟) = 距离(AU) ÷ 跃迁速度(AU/分钟)
跃迁速度 = 舰船基础速度 × (1 + 跃迁引擎操控等级 × 10%)
```

### 挖矿机制
```
挖矿速度 = 基础速度 × (1 + 采矿技术等级 × 10%)
```
- 矿舱满后自动回行星卸货（每次卸货暂停1分钟）
- 蜜蜂级采矿护卫舰：矿舱500m³，速度1m³/秒
- 蝗虫级采矿驳船：矿舱3000m³，速度5m³/秒

### 运输船货仓
| 舰船 | 基础容量 | 货运管理加成 |
|------|----------|--------------|
| 崆峒级运输舰 | 60000m³ | 每级+10% |
| 泰山级货舰 | 2000000m³ | 每级+10% |

- 超载处理：自动按剩余空间装载，优先装载体积小的矿物
