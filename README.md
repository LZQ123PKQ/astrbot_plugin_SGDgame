# 🚀 星际黎明 (Star Dawn)

一款类似 EVE Online 的 AstrBot 太空挂机游戏插件。

> 在浩瀚星海中建立你的帝国，从一名普通飞行员成长为星际传奇。

## 游戏特色

- 🌌 **星系探索**：多个星域、星座、星系，自由跃迁
- ⛏️ **挂机挖矿**：多种矿石可开采，矿舱容量限制
- 👾 **刷怪系统**：自动战斗，获取赏金
- 🔥 **精炼系统**：将原矿提炼成矿物
- 🔧 **制造系统**：消耗矿物制造舰船
- 🚛 **运输系统**：自动运输物品，支持货柜和矿舱管理
- 💰 **市场系统**：吉他市场交易，支持买卖订单
- 🗺️ **星图导航**：查看星域、星座、星系结构，自动规划路线

## 安装

1. 将插件文件夹复制到 AstrBot 的插件目录
2. 重启 AstrBot 或重载插件
3. 发送 `/游戏注册` 开始游戏

## 游戏机制

### 初始设置
- 出生点：吉他
- 初始舰船：冲锋者级采矿护卫舰
- 初始资金：¥0

### 挖矿机制
- 矿舱满后自动回空间站卸货
- 冲锋者级：矿舱5000m³，挖矿速度8.51m³/s
- 回旋者级：矿舱27500m³，挖矿速度20.96m³/s

### 刷怪机制
- 根据舰船类型和安等等级自动匹配合适的刷怪等级
- 等级1-9，等级越高赏金越高但风险越大
- 安等越低，可刷等级越高

### 导航机制
- 自动规划最短路径
- 考虑星门连接关系
- 可实时查看导航进度

### 运输机制
- 自动装载-导航-卸载循环
- 支持货柜和矿舱
- 可查看运输状态和进度

### 市场机制
- 仅吉他星系支持市场交易
- 卖单：冻结物品，成交后获得货币
- 买单：冻结货币，成交后获得物品
- 支持部分成交
- 取消订单返还剩余冻结物品/货币

### 安等等级说明
- **高安（0.5 ~ 1.0）**：安全区域，适合挖矿，主要分布在伏尔戈、赛塔德洱、辛迪加、长征等星域
- **低安（0.0 ~ 0.4）**：危险区域，适合刷怪，主要分布在美特伯里斯星域
- **00区（< 0.0）**：危险区域，高收益，主要分布在静寂谷、对舞之域、琉蓝之穹等星域
